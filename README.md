# ChuckNorrisJoke

## Описание 
Андроид приложение позволяет получать по сетевому запросу шутку Чака Норриса по выбранной категории.

## Технологии 
Клиентская часть: android studio kotlin. Запрос api

- clean,<br/>
- viewModel,<br/>
- mvvm, <br/>
- di hilt, <br/>
- navigation, <br/>
- coroutine, <br/>
- databinding, <br/>
- retrofit, <br/>
- animation gif,<br/>
- fragment, <br/>
- recyclerView, adapter
