# ChuckNorrisJoke

## Описание 
Андроид приложение позволяет получать по сетевому запросу шутку Чака Норриса по выбранной категории.

## Работа с кодом 
Клиентская часть: android studio kotlin. Запрос api

- viewModel,<br/>
- mvvm, <br/>
- coroutine, <br/>
- databinding, <br/>
- animation gif,<br/>
- retrofit<br/>
